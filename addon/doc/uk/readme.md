# Ігнорувати промовляння виділеного.

Цей додаток додає поточне виділення до тимчасового словника, замінюючи його порожнім рядком. Таким чином, виділене буде пропущено у мовленні. Корисно для читання текстів з повторюваними або непотрібними словами/рядками.

Додані записи будуть видалені при перезапуску NVDA або видаленні з записів у тимчасовому словнику.
